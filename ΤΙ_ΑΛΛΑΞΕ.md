## Το κουμπί «Πωλήσεις»

**Χωρίς GitHub token:** σου δίνει σύνδεσμο για να το τρέξεις χειροκίνητα.

**Με token:** ένα κλικ και τρέχει.

### Πώς να φτιάξεις το token

1. GitHub → Settings → Developer settings → **Fine-grained tokens** → Generate

   | Πεδίο | Τιμή |
   |---|---|
   | Repository access | Only select → `ab-skyros-dashboard` |
   | Permissions → **Actions** | **Read and write** |

2. Streamlit Cloud → Settings → Secrets:

   ```toml
   GITHUB_TOKEN = "github_pat_..."
   GITHUB_REPO  = "abskyros/ab-skyros-dashboard"
   ```
